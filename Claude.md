# PPT翻訳アプリの最新状況

- PPTXファイルのアップロード機能
- スライドのプレビュー表示
- テキスト抽出と翻訳機能
- 翻訳結果のダウンロード機能
- ユーザー認証システム
- チーム管理機能
- アクティビティログ
